Jconcole

  1. 修改远程启动脚本tomcat/bin/startup.sh中增加一下变量： 
        CATALINA_OPTS="-Djava.rmi.server.hostname=192.168.82.88 \
                   -Dcom.sun.management.jmxremote \
                   -Dcom.sun.management.jmxremote.port= 10900 \
                   -Dcom.sun.management.jmxremote.ssl=false \
                   -Dcom.sun.management.jmxremote.authenticate=false"
    export CATALINA_OPTS
  2. 其中-Djava.rmi.server.hostname项必须设置，否则远程连接会因为解析到127.0.0.1失败，该项的值就是你在windows客户端连接linux时的ip地址
  3. ssl和authenticate先设置为false，如果需要安全，请不要false
  4. 本地运行 
        jconsole 192.168.83.88:10900
