FileChannel

  1. 建立文件hello.txt
  2. 进程A逐条写入100万条
  3. 进程B一条条读出
  4. 进程A,B同时启动, 一个写，一个读
  5. 读写期间, 重命名：mv hello.txt world.txt;
  6. 读写期间，重建新的hello.txt: echo “hello” > hello.txt

结果：

  • 总数据量：2.1G
  • 进程A写耗时：297427ms
  • 进程B写耗时：297599ms

通过 文件+Rotate+FileChannel 快算交换数据也许不错。
