设置系统参数fs.inotify.max_user_watches

日期: 2012-04-20 10:04

  • 设置系统参数fs.inotify.max_user_watches
    • 临时修改
    • 修改系统配置

临时修改

  echo 100000 | sudo tee /proc/sys/fs/inotify/max_user_watches

修改系统配置

  1. 修改配置文件. 在配置文件/etc/sysctl.conf中加入： 
      fs.inotify.max_user_watches = 524288
  2. 使配置生效 
      sudo sysctl -p
