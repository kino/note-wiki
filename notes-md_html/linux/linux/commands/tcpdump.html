Commands

限制抓包个数

 tcpdump -c100

写入文件

 tcpdump -c100 -w o.dump
 tcpdump -c100 | tee o.dump

指定设备

 tcpdump -i eth0

过滤地址

 tcpdump host 10.2.6.201
 tcpdump src 10.2.6.201
 tcpdump dst 10.2.6.201
 # 过滤网络
 tcpdump net 10.2.6

过滤协议

tcpdump ip
tcpdump tcp
tcpdump udp

过滤size

tcpdump greater 200

过滤端口

tcpdump port 8080
